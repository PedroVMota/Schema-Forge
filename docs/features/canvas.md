# Visual Canvas

The canvas is the core of Schema Forge — an interactive workspace where you design your database schema.

<!-- TODO: Add screenshot -->
![Canvas Overview](../assets/screenshots/canvas-overview.png)

## Navigation

- **Pan** — Click and drag on the background
- **Zoom** — Scroll wheel or pinch gesture
- **Fit view** — Double-click the background to fit all tables in view

## Minimap

A minimap in the bottom-right corner gives you an overview of your entire schema. Click on the minimap to jump to a specific area.

## Toolbar

The top toolbar provides quick access to:

| Action | Description |
|--------|-------------|
| Add Table | Create a new table on the canvas |
| SQL Editor | Toggle the SQL editor panel |
| Export SQL | Copy generated SQL to clipboard |
| Import Prisma | Open the Prisma import modal |
| Theme | Switch color theme |
| Stats | View table and relation counts |

## Canvas Controls

Use the controls in the bottom-left corner to:

- Zoom in / out
- Fit all nodes to view
- Toggle interactive mode (lock/unlock)
