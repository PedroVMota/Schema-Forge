# Features

Schema Forge is packed with tools to make database design fast and intuitive.

## Core Features

### [Visual Canvas](./canvas.md)
Interactive drag-and-drop canvas for designing schemas visually. Zoom, pan, minimap, and more.

### [Table Editor](./tables.md)
Create and edit tables with full control over columns, types, and constraints.

### [Relationships](./relationships.md)
Draw foreign key relationships between tables by connecting columns.

### [SQL Import & Export](./sql.md)
Import existing `CREATE TABLE` statements or export your visual design as SQL.

### [Prisma Import](./prisma.md)
Import Prisma schema files and instantly visualize them as an ERD.

### [Projects](./projects.md)
Organize your work into multiple projects, all saved locally in your browser.

### [Themes](./themes.md)
Switch between dark and light modes with multiple color schemes.
