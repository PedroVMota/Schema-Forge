# Introduction

## What is Schema Forge?

Schema Forge is a free, open-source visual database schema editor. It lets you design database schemas using an interactive canvas — create tables, define columns, draw relationships, and export production-ready SQL. All from your browser, with zero backend required.

## Why Schema Forge?

- **No sign-up required** — open the app and start designing
- **Runs entirely in your browser** — your data never leaves your machine
- **Import SQL or Prisma** — paste existing schemas and instantly visualize them
- **Export SQL** — generate CREATE TABLE statements from your visual design
- **Multiple projects** — organize schemas into separate projects with local persistence
- **Dark & light themes** — with multiple color schemes to choose from
- **Open source** — free forever, community-driven

## Who is it for?

- **Developers** prototyping a new database schema
- **Students** learning about relational databases and ERD diagrams
- **Teams** who want a quick, shareable visual representation of their data model
- **Anyone** who prefers visual tools over writing raw DDL

## How it works

1. **Create tables** on the canvas or import from SQL/Prisma
2. **Define columns** with types, constraints (PK, FK, unique, nullable)
3. **Draw relationships** between tables by connecting columns
4. **Export SQL** when your design is ready

## Browser Support

Schema Forge works in all modern browsers:

- Chrome / Edge (latest)
- Firefox (latest)
- Safari (latest)

Data is stored in your browser's `localStorage`, so your projects persist between sessions on the same browser.
